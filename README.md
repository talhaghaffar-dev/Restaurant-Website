# Restaurant Website

>  Just finished building a modern and responsive Restaurant Website.

This project showcases a premium restaurant website with a clean user interface, smooth navigation, and a fully responsive design. It is designed to provide an engaging user experience for customers on both desktop and mobile devices.

## ✨ Features

- 🍕 Modern Hero Section
- 📱 Fully Responsive Design
- 💬 WhatsApp Integration
- 📞 Click-to-Call Button
- 📍 Google Maps Integration
- 🍔 Interactive Menu Section
- 🖼️ Food Gallery
- ⭐ Customer Reviews
- 📩 Contact Section
- 🎨 Premium UI & Smooth Animations

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📚 What I Learned

This project helped me improve my frontend development skills, responsive web design, UI/UX design, and building real-world business websites.
